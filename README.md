# Web Programlama II - Dönem Ödevi

Bu depo (repository), **Sivas Cumhuriyet Üniversitesi - Bilişim Sistemleri ve Teknolojileri** bölümü Web Programlama II dersi dönem ödevi kapsamında oluşturulmuştur.

## Projenin Amacı
Projenin temel amacı, **Django** web framework'ünün resmi dokümantasyonunda yer alan **Signals (Sinyaller)** konusunun Türkçeye çevrilmesi ve bu konunun daha iyi anlaşılabilmesi için proje ortakları tarafından bağımsız, küçük örnek uygulamalar (projeler) geliştirilmesidir.
